# <img src="assets/logo.svg" width="80" height="80" align="center"> Danaa (دانـا)

**Danaa** is an AI-powered Telegram bot designed to assist Iranian immigrants in Canada. It leverages community knowledge to provide context-aware, privacy-conscious information about life and immigration in Canada.

## 🚀 Vision
To provide a reliable, always-available "knowledge companion" that helps immigrants navigate the complexities of life in Canada by drawing on the collective experience of their community.

## 🏗️ Project Status: Phase 1 (Foundation)
Current Focus: Building the core Telegram bot infrastructure.

## 🛠️ Tech Stack (Initial)
- **Backend:** Python (FastAPI)
- **API:** Telegram Bot API
- **Deployment:** AWS Lambda (Planned)

## ⚖️ Disclaimer
*Danaa provides information based on community history; it does not provide legal or professional advice. Information may be outdated or inaccurate; verify with official sources.*
